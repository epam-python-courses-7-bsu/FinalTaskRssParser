## JSON Structure

```json
[
  {
    "feed":
      {
          "title": "Title of the channel",
          "description": "Channel description"
      },
      "items": [
        {
          "title": "Title of the linked article",
          "link": "Link to the full article",
          "image_links": "Links to images contained in article",
          "description": "Description of the info linked in item element",
          "pub_date":  "Date when article was published"
        }
      ]
  }
]
```
